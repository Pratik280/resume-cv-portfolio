# Top 25 Java Coding Interview Questions (Master List – Backend, 4 YOE)

Covers service companies (TCS, Infosys, Accenture, HCL, Deloitte, EPAM) and product companies.

---

## 1. Arrays

- Find the largest element in an array
- Find the second largest element in an array without sorting
- Find the smallest element in an array
- Find duplicate elements in an array
- Move all zeros to the end while maintaining order
- Find the missing number in an array containing numbers from 1 to N
- Find intersection of two arrays
- Find maximum subarray sum (Kadane’s algorithm)

---

## 2. Strings

- Reverse a string without using built-in reverse function
- Reverse words in a sentence without using library reverse
- Check if two strings are anagrams
- Find first non-repeating character in a string
- Find duplicate characters in a string
- Count frequency of characters in a string using HashMap
- Find longest substring without repeating characters

---

## 3. Numbers / Math Logic

- Check if a number is prime efficiently
- Reverse an integer without converting to string
- Check if a number is palindrome
- Find factorial of a number
- Find GCD and LCM of two numbers
- Check if a number is Armstrong number

---

## 4. Collections / HashMap / Set

- Count frequency of each element in a list using HashMap
- Find first non-repeating element in a list
- Find duplicate elements in a list using Set
- Find most frequent element in a list
- Group list of anagrams together using Map

---

## 5. Java Streams (VERY IMPORTANT)

- Find second highest number in a list using Streams
- Find duplicate elements using Streams
- Find first non-repeating element using Streams
- Find elements appearing more than K times using Streams
- Partition list into even and odd numbers using Streams

---

## 6. Object-Based / Real Backend Problems (MOST IMPORTANT FOR 4 YOE)

- Find second highest salary from employee list
- Find duplicate objects based on a field
- Find first non-repeating object from list
- Group employees by department using groupingBy
- Sort list of objects by multiple fields using Comparator and Streams
